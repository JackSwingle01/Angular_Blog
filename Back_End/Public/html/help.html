Help Page

Route Method Functionality Auth Required?
[GET] / Returns this help page. No
[GET] /Users Returns a list of users. Yes
[GET] /Users/{id} Returns a single user with the specified id. Yes
[POST] /Users Creates a user specified by a json object.
Must be formatted as follows, all fields required:
{
"userId": "___",
"firstName": "___",
"lastName": "___",
"emailAddress": "___",
"password": "___"
} 		No
[PATCH] /Users/{id} Edits an existing user with specified id.
Id cannot be changed. Any number of attributes may be included or ommited. 		Yes
[DELETE] /Users/id Deletes the user with the specified id. Yes
[GET] /Users/{userId}/{password} Logs in user. No
[GET] /Posts/User/{userId} Returns all the posts for a given user. No
[GET] /Posts Returns all the posts in order of creation date. No
[GET] /Posts/{postId} Returns a specific post. No
[POST] /Posts Creates a new post for the current user. Yes
[PATCH] /Posts/postID Updates an existing post. Only the title, content, and headerImage may be edited. Updates the lastUpdated value. Yes
[DELETE] /Posts/{postId} Deletes a post. Yes
